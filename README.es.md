Esta es una herramienta **sin ánimo de lucro** para seleccionar aleatoriamente un personaje de Overwatch. Las imágenes incluidas en ella se obtuvieron de la página oficial de [Overwatch](https://overwatch.blizzard.com/es-es/heroes/) y son propiedad de [Blizzard](https://www.blizzard.com/es-es/).

## Consigue Overwatch Roulette

### 🌐 Jugar en Vercel (Recomendado)
Este proyecto está publicado en [Vercel](https://vercel.com/). 
Simplemente navega a [Overwatch Roulette](https://overwatch-roulette.vercel.app/) ¡y disfruta!

### 💻 Desde GitHub (Versión local)
Clona el repositorio en tu máquina local: 

```bash
git clone https://github.com/formatocd/overwatch-roulette.git
```

Después, simplemente abre el archivo `index.html` en tu navegador web favorito.

### 🐳 Desde Docker Hub (Contenedor)
Si quieres usar Docker (o Podman), ejecuta un contenedor con este comando:

```bash
docker run -p 9000:80 -d formatocd/overwatch-roulette
```

Una vez que el contenedor inicie, navega a `http://localhost:9000` en tu navegador.
